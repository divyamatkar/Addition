AJAX – Fetching Remote Data using XMLHttpRequest

Aim:

To create a program using AJAX and the XMLHttpRequest object to fetch and display remote data from a server without reloading the webpage.

Objectives:

  • To understand how AJAX works in asynchronous web communication.
  • To learn how to send HTTP requests using the XMLHttpRequest object.
  • To dynamically fetch and display data from a remote server.
  • To demonstrate background data loading without refreshing the webpage.

Theory:

AJAX (Asynchronous JavaScript and XML) is a technique used in web development to send and receive data from a server asynchronously without requiring a full page reload. AJAX improves user experience by allowing parts of a web page to update dynamically.

XMLHttpRequest Object:

  • This is a built-in JavaScript object used to interact with remote servers.
  • It allows sending HTTP requests such as GET, POST, PUT, and DELETE.

Working Process:

  1. Create an XMLHttpRequest object.
  2. Open the connection using xhr.open().
  3. Send the request using xhr.send().
  4. Receive and process the response when readyState === 4 and status === 200.

Advantages of AJAX:

  • Faster and more responsive webpages.
  • Reduces server load.
  • Provides better user experience.

Conclusion:

By creating this AJAX program using XMLHttpRequest, we learned how to fetch remote data asynchronously and display it on a webpage without reloading the page. This practical demonstrates how AJAX enhances web interactivity and improves dynamic content handling in modern web applications.
