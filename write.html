Form Validation using JavaScript

Aim:

To create a JavaScript-based form validation program that validates user input fields such as name, email, password, and confirm password before submitting the form.

Objectives:

  • To understand how to validate form fields using JavaScript on the client side.
  • To check mandatory fields like name, email, password, and confirm password.
  • To apply validation rules such as minimum length, pattern matching, and password matching.
  • To display appropriate error and success messages dynamically on the webpage.

Theory:

Form Validation:

Form validation is the process of checking whether the data entered by the user in a form is correct, complete, and acceptable before it is sent to the server.

  • Client-side validation: Done using JavaScript in the browser.
  • Server-side validation: Done on the server after submission.

Client-Side Validation using JavaScript:

JavaScript is commonly used because it:

  • Provides instant feedback to the user.
  • Reduces unnecessary server requests.
  • Improves user experience.

Validation Rules Used:

Name:

  • Cannot be empty.
  • Must contain only letters and spaces (/^[A-Za-z\s]+$/).
  • Minimum length 3 characters.

Email:

Validated using regular expression: ^[^\s@]+@[^\s@]+\.[^\s@]+$

Password:

  • Must be at least 6 characters long.

Confirm Password:

  • Must match the password field.

DOM Manipulation:

Uses document.getElementById() to read values and display messages dynamically.

Event Handling:

Form submission handled using:

document.getElementById('myForm').addEventListener('submit', ...);

event.preventDefault() stops form submission when validation fails.

Conclusion:

This practical helped us understand JavaScript form validation, DOM manipulation, event handling, and how validation improves user experience by preventing invalid data entry.


Steps to Set Up Docker Container Environment

  1. Install Docker Desktop
    • Download and install Docker from the official website.
    • Start Docker and ensure it is running.
  2. Verify Installation

    Run command:

    docker --version
  3. (Optional) Install NVIDIA Docker (for GPU support)
    • Install NVIDIA GPU drivers.
    • Install NVIDIA Toolkit:
    sudo apt install nvidia-container-toolkit
sudo systemctl restart docker
  4. Pull a Docker Image

    Examples:

    docker pull ubuntu

    Or GPU image:

    docker pull nvidia/cuda
  5. Run a Container

    Normal:

    docker run -it ubuntu

    With GPU:

    docker run --gpus all -it nvidia/cuda
  6. Check Running Containers 
    docker ps
  7. Stop / Remove Container 
    docker stop <container-id>
docker rm <container-id>
