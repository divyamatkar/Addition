Addition Program Using JavaScript

Aim:

To create a JavaScript program that performs the addition of two numbers by taking user input and displaying the result on a webpage.

Objectives:

  • To understand how to take input values from the user using HTML input elements.
  • To apply JavaScript for performing arithmetic operations.
  • To dynamically display the output using DOM manipulation.
  • To enhance user interaction through a graphical interface.

Theory:

JavaScript is a client-side scripting language used to make web pages interactive. Addition is a basic arithmetic operation where two numeric values are summed to get a final result.

To build this program, the following concepts are used:

HTML Input Fields:

<input type="number"> is used to allow users to enter numeric values.

JavaScript Variables and Operators:

Variables are used to store the values entered by the user, and the + operator is used to perform the addition operation.

DOM Manipulation:

The method document.getElementById() is used to fetch input values and display the result dynamically without refreshing the webpage.

Example calculation:
If number1 = 10 and number2 = 5 → Result = 10 + 5 = 15

This demonstrates how JavaScript connects with HTML to create interactive applications.

Conclusion:

By performing this program, we successfully learned how to accept user input and process arithmetic operations using JavaScript. This practical helped us understand DOM manipulation, event handling, and how JavaScript enhances webpage interactivity by showing results instantly.
